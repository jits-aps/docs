TList << Click to Display Table of Contents >> Navigation:  Script Reference Guide > Objects > TList   Description TList, which stores an array of pointers, is often used to maintain lists of objects. TList introduces properties and methods to:   •Add or delete the objects in the list•Rearrange the objects in the list•Locate and access objects in the list•Sort the objects in the list.  Object Hierarchy TList - TObject   Properties of TList   Name Description Count Indicates the number of entries in the list that are in use. Items Lists the object references, where 0 is the first item, 1 is the second item, and so on.   Methods of TList   Name Description Add Inserts a new item at the end of the list. Assign Copies elements of one list to another. Clear Deletes all items from the list. Delete Removes the item at the position given by the Index parameter. IndexOf Returns the index of the first entry in the Items array with a specified value. Insert Adds an object to the Items array at the position specified by Index. Move Changes the position of an item in the Items array. Remove Deletes the first reference to the Item parameter from the Items array.
