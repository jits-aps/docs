TEmail << Click to Display Table of Contents >> Navigation:  Script Reference Guide > Objects > TEmail   Description Use TEmail to create an email message and send it. This component works with any default mail programs which support MAPI. For example, MS Outlook, MS Outlook Express etc.   Example   procedure SpeedButton1Click(Sender: TObject); var   e: TEmail; begin   e := TEmail.Create;   try     e.Recipients.Add('info@admind.dk');     e.Subject := 'E-mail titel';     e.Body := 'This is a test';     e.Send;   finally     e.Free;   end; end;   Object Hierarchy TEmail - TComponent - TPersistent - TObject   Properties of TEmail   Name Description AttachedFiles Use this TStringList property to specify attachments. BlindCopyTo Use this TStringList property to add carbon recipients. If you want the user to enter the recipients, ignore this property and set ShowDialog to True. Body Use this string property to enter the message. If you want the user to enter the message, ignore this property and set ShowDialog to True. CopyTo Use this TStringList property to add carbon recipients. If you want the user to enter the recipients, ignore this property and set ShowDialog to True. FromAddress Use this property if you want to specify a from address associated with the e-mail. FromName Use this property if you want to specify a sender name associated with the e-mail. Recipients Use this TStringList property to add the recipients. If you want the user to enter the recipients, ignore this property and set ShowDialog to True. ShowDialog This option determines if an e-mail dialog window should appear by the default mail program. Otherwise the mail will be sent automatically. Subject Use this string property to enter a mail titel.   Methods of TEmail   Name Description Reset This procedure will clear all properties. Send This procedure will send the mail. If the property ShowDialog is set to false, you must specify a recipient, a subject and a text message, before you execute this method.
