TProgressDialog << Click to Display Table of Contents >> Navigation:  Script Reference Guide > Objects > TProgressDialog   Description TProgressDialog shows a standard window with a TProgressBar. Progress bars provide users with visual feedback about the progress of a procedure.   Example   var   i: Integer; begin   with TProgressDialog.Create do   try     Caption := 'Form title';     Text := 'Checking records...';     Min := 1;     Max := DataSet.RecCount - 1;     while not DataSet.Eof do     begin       Position := Position + 1;       // Check if user has pressed the Cancel button       ProcessMessages;       if ModalResult = mrCancel then         Abort;       // Execute the procedure per record       CheckCustomer(DataSet.FieldByName('No').AsString);       DataSet.Next;     end;   finally     Free;   end;   Object Hierarchy TProgressDialog - TComponent - TPersistent - TObject   Properties of TProgressDialog   Name Description AllowCancel Controls whether the cancel button responds to mouse and keyboard. Set AllowCancel to False if you want to disable the Cancel button. Note! This feature has been implemented in version 5.30.01 Caption Use Caption to specify the text string that labels the form. Min Max Specifies the lower (Min) and upper (Max) limit of the range of possible progress positions.   Use Min along with the Max property to establish the range of possible positions of the progress bar. When the process tracked by the progress bar is complete, the value of Position should equal Max. ModalResult Use this property to find out if the user has pressed the Cancel button. Returns mrCancel if the buttons has been pressed. Otherwise it return mrNone (0).   Note! Call ProgressMessages before checke the ModalResult. ProcessMessages permit the application to handle user activities. (from the Windows Messages Queue) Position Specifies the current position of the progress bar. Text Use this string property to enter the message. If you want the user to enter the message, ignore this property and set ShowDialog to True.   Methods of TProgressDialog   Name Description Create Creates and initializes a TProgressDialog instance. Free Destroys the TProgressDialog and frees its memory.
