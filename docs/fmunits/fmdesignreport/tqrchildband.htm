TQRChildBand << Click to Display Table of Contents >> Navigation:  Design Units > Rapport Designer > TQRChildBand General Use the TQRChildBand component to create a band that sticks to the bottom of another band. This is useful if you want some controls to move down depending on the vertical expansion of some other controls. Place the expanding controls on one band and put any controls that should move down on a Child band.   If you want to make sure the two bands always print on the same page use the LinkBand property to connect them. The easiest way to add a child band is via the Design Menu. Right click on the band who need a child band, and select Has Child.   Object Hierarchy TQRChildBand - TQRCustomBand - TCustomControl - TWinControl - TControl - TComponent - TPersistent - TObject   Properties of TQRChildBand   Name Description AlignToBottom Use AlignToBottom to force a band to print at the bottom of a page instead of at the top.
