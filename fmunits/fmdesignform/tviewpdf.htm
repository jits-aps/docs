Width Specifies the horizontal size of the control or form in pixels.   Methods of TViewPDF   Name Description FindText This method searchs for the substring "Text" in the document. When found, it returns True.  and select substring and scroll to it. If FromStart is True, the search starts from the beginning of the document. If CaseSensitive is True (default = False), a character case is taken into account when comparing strings. if HighLight is True (default), the found text will be highlighted. LoadFromFile Loads document from a file. LoadFromStream Loads document from a Stream. Print Print the loaded document. Use the parameter Range to choose which pages to print. If not empty, this string is interpreted as page range, i.e. "1-3,7,15". It is also possible to specify the number of copies with "*", i.e. "1-3*2" to print two copies of the pages 1 to 3. PrintDialog Display the print dialog.   Events of TViewPDF   Name Description OnChangeViewPage Occurs after a new page is displayed. Use the parameter PageNo to get the new page number. OnDblClick Occurs when the user double-clicks the left mouse button when the mouse pointer is over the control. OnEnter Occurs when the control receives the input focus. Use the OnEnter event handler to cause any special processing to occur when a control becomes active. OnError Occurs when the control cannot show the document. Use the parameter Msg to get the error message. OnExit Occurs when the input focus shifts away from the control to another control. Use the OnExit event handler to provide special processing when the control ceases to be active. OnKeyDown Occurs when a user presses any key while the control has focus.   Use the OnKeyDown event handler to specify special processing to occur when a key is pressed. The OnKeyDown handler can respond to keyboard keys, including function keys and keys combined with the SHIFT, ALT, and CTRL keys, and pressed mouse buttons. See TShiftState for a description of the Shift parameter. OnKeyPress Occurs when a key is pressed. OnKeyUp Occurs when the user releases a key that has been pressed. OnMouseDown Occurs when the user presses a mouse button with the mouse pointer over a control. OnMouseMove Occurs when the user moves the mouse pointer while the mouse pointer is over a control. OnMouseUp Occurs when the user releases a mouse button that was pressed with the mouse pointer over a component. OnNeedPassword Occurs when loading an encrypted PDF file and need a password. Define the password by the parameter Password.
