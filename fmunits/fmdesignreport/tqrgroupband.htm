TQRGroupBand << Click to Display Table of Contents >> Navigation:  Design Units > Rapport Designer > TQRGroupBand General The QRGroupBand component is similar to a header band, except it has an associated numerical expression whose value causes the group to break. A footer band can be associated with a group control so that the group can have a header and a footer.   The group expression is evaluated on every record and if it changes from the previous value, the group âbreaksâ - that is the group header is printed.
