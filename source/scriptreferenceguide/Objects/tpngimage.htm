TPNGImage << Click to Display Table of Contents >> Navigation:  Script Reference Guide > Objects > TPNGImage   Description TPngImage encapsulates a PNG (Portable Network Graphics) graphic.   Use TPngImage to read and write PNG graphics. A PNG image supports transparency and a variety of compression options. The object has no canvas (so it cannot be drawn upon). However, load a PNG file into a TImage component and let the TImage component display the PNG.   Example   procedure FormCreate(Sender: TObject); var   PNG: TPNGImage; begin   PNG := TPNGImage.Create;   try     PNG.LoadFromFile('c:\test.png');     Image1.Picture.Assign(PNG);   finally     PNG.Free;   end; end;   Object Hierarchy TPNGImage - TGraphic - TPersistent - TObject   Properties of TPNGImage   Name Description Empty Indicates whether the graphics object contains a graphic. FileExt Returns the current file extension identifier. (png) Height Specifies the vertical size of the graphic in pixels. Width Determines the maximum width of the graphics object in pixels.   Methods of TPNGImage   Name Description Assign Copies or converts an image object to a PNG object. LoadFromFile Loads the PNG image from a file. LoadFromStream Loads the PNG image from a TStream object. SaveToFile Saves the PNG data to a file. SaveToStream Saves the PNG data to a TStream object.
